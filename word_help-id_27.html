水墨印刷技术在瓦楞纸箱中印刷适性及应用问题解析

2018-05-02 14:19:23
中国纸箱网

影响瓦楞纸箱印刷质量的因素有很多，如印刷设备、印刷版材、承印底材、印刷压力以及水性油墨的印刷适性等等。那么水墨印刷技术作为其中关键的因素之一，如何掌握水墨的相关特性和水墨印刷的应用技巧就显得非常重要。

水墨的物理特性和水墨印刷适性在应用中的关系

１、颜色：印刷的最终目的是实现颜色的再现。

n 色强度：在印刷中影响色强度的主要因素是通过传墨装置转移到印版的油墨厚度，即油墨的转移量。

n 颜色的三属性分别是色相、亮度和纯度。

色相：是反应颜色固有的色彩相貌的，是色与色之间的主要区别。

亮度：是反映色彩明暗程度的。

纯度：即饱和度。是反应色彩接近自然色光的程度。

n 影响颜色的主要因素

因为颜色的外观存在着透色率、反应率和同分异物现象（即在某同一光源的条件下，两种颜色非常匹配，而在另一种光源下却不匹配的现象）。颜色会受观察条件的不同变量的影响（如观察角度、照明程度、光泽油墨的不透明度和叠印图文等）。

承印物的表面状态对油墨的显色有影响。如光洁度、吸墨性、表面颜色等。

墨层的厚薄和色浓度等问题会引起颜色深浅的变化。墨层的干燥时间对颜色的观察和测定都有影响。因为油墨干燥前，墨层表面对反射透射及吸光性和干燥后是不同的。深色油墨干燥后颜色更深些，而浅色油墨干燥后颜色则更浅些，这是湿墨和干墨的变化规律。

n 颜色的测定

光（灯）源：因为有光才有色，光的波长不同，即便是同一颜色在不同光的作用下反映出的颜色也是有区别的，所以检测颜色时，最好使用标准色光源。一般标准光源有D65和D50等，因它在可见光谱范围内（360-750纳米）具有红、绿、蓝能量相等的特性，使人容易观察色彩的轻微差异。因此，如果对颜色的要求严格时，使用标准光源是很重要的。

电脑检测系统，因为不同的颜色具有各自的波长，电脑就是根据感受到的波长来检定出颜色的，一般电脑软片为CIEL.A.B系统（L代表亮度颜色中白至黑范围即明暗程度。A指颜色中红至绿的范围。B指颜色中蓝至黄的范围）。根据测定颜色的L.A.B值只能测定颜色的色相和亮度，而不能测定颜色的纯度，（前文提到的颜色的三属性）所以实际颜色检验中仍有10％-20％的误差，若配上L.C.H系统，则能测出的颜色的纯度则误差只有5％-10％，测出△E的话，误差仅为3％左右。因此，只要有完整的检测系统就能测出高品质的颜色。

２、粘度：油墨内聚力的大小，是水墨的主要指标之一。

粘度的测定：

对于粘度的测量工具是粘度环和秒表。其检测的方法是用规定容量和漏孔孔径的粘度杯，在粘度杯中盛满水墨检测所流出的水墨直至其断线时所需要的时间（用秒表计算）。因为目前应用的粘度杯规格不尽相同，种类较多，所以不同规格的粘度杯所测出的结果都是不一样的。目前，对于水性油墨的测量粘度杯多数采用日本的RIGOSHA4#CUP和国内的涂4#号杯等。

水性油墨粘度的控制范围

n 一般实地印刷使用水墨的粘度在15秒-25（RIGOSHA4#CUP）秒之间，该粘度范围的水墨平流性好，实地印刷适应性好。若粘度过高，油墨容易脏版令印品产生毛边；若粘度过小，影响载体带动色料的能力，实地印刷有水纹漏白或色强度不够等现象。

n 网线和小字印刷使用水墨粘度在8秒-15秒（RIGOSHA4#CUP）之间为好，该粘度范围的水墨流动性好，对网点的再现性能强，印刷出的产品网点清晰，层次分明，变形小，不易脏版粘版。

n 多色印刷产品应该满足套印不漏白的前提下，合理的安排好印刷色序。控制好前后色序的上机粘度和墨层的干燥时间，保证后一色印刷的墨色能将前一色遮盖住。

如上所述，粘度的大小直接能影响印刷品的质量。但由于印刷条件的不尽相同。对上机粘度的要求也不一样，为了更精确的控制好水墨的上机粘度，应该在印刷过程中定时检验水墨的粘度。结合不同粘度所印刷出的产品效果，记录其最佳数值，保持以此最佳数值范围的粘度进行印刷。

３、干燥也是水墨较为重要的指标之一，因为干燥的快慢和粘度一样能直接表现在印刷的品质上。

n 干燥原理

一般的水墨的干燥包括挥发、吸收和反应性固化三种方式。操作人员必须详细了解水墨的干燥原理，才能根据产品或承印物的不同，合理的调配水墨的干燥时间。

n 干燥速度

干燥过快，墨层容易结皮，水墨易干附在印版上，造成脏版粘版，印刷不饱满和网点不清，印刷品起毛边等现象，解决此问题可在水墨中适量的添加慢干剂调整；干燥过慢，容易发生粘连，使印刷品蹭脏托花，甚至影响叠色印刷，该问题可根据PH值的高低，适量添加快干剂调整或选用较快干的水墨印刷。

PH值

水性墨中含有一定量的胺溶剂，用于控制和提高水墨的稳定性。因此，PH值是保证水墨适应性的重要指标之一。水墨出厂的PH值一般控制在8.2?.2（25℃）之间。使用时根据实际需要，如承印物或者温度的不同将其控制在8.0?.5之间。PH值的测量工具是PH计和温度计。由于水墨中的胺溶液的挥发和承印物的酸碱性的影响，在印刷过程中PH值会有所变化，建议操作人员应定时（最好2?小时内）对水墨的PH值进行检测。适量的添加PH稳定剂对其进行调整，确保水墨稳定的印刷适性。
