纸板度数如何计算及怎么采购划算

2018-05-08 00:22:24
中国纸箱网

纸板度数计算公式:（宽+高）x N / 2.54 = Y

公式中“N”代表纸板开数，如宽+高数值较大，建议开一，N值就不用计算了。

公式中“Y”代表纸板度数，这值根据就近原则，求大即可。

例如：（30+20）x 2 / 2.54 = 39.37″ 根据纸箱就近采购原则，此次得采购41″的纸板。

从原则上来说，一片箱最少，因为少了一个钉条位（驳口、粘接纸舌），纸板一开多张（个）比一开一更省，由于有两个或多个边用是分纸刀分开的，所以这两个边的边料就省了，一般来说，尽量一开多，但得考虑分纸机最大分纸幅度，以及纸板长度，适中的纸板长度，一开多，不仅节省了分纸师傅的分纸次数，更提高了生产效率，减低了浪费。
